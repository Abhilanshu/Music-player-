# 🎵 Muse Music Player

> A Premium, Spotify-like Music Streaming Experience. Built for high performance, mobile-first use, and ultimate discovery.

![Muse Preview](https://via.placeholder.com/1200x600?text=Muse+Music+Player+Preview)

## ✨ Features

- **🚀 Lag-Free Performance:** Optimized React architecture for a smooth 60fps experience.
- **🔍 Semantic Search:** Category-based search (Songs, Playlists, Albums) with side-panel details.
- **🔄 YouTube Fallback:** Never hit a dead end. Automatic fallback to YouTube for rare tracks.
- **💎 Premium Audio:** 
    - **5-Band Equalizer** for custom sound tuning.
    - **Crossfade** for seamless transitions.
    - **High-Res Audio** support (320kbps).
- **📱 PWA & Mobile First:** Install as an app. Swipe gestures, haptics, and responsive layout.
- **🚗 Car Mode:** A simplified, high-contrast interface for safe driving.
- **💾 Offline Mode:** Cache your favorite tracks for listening without data.
- **🎨 Immersive Visuals:** Fullscreen Canvas mode with dynamic animated backgrounds.

## 🛠️ Architecture

- **Core:** React 19 + Vite 8
- **State:** Context API (Hybrid State Management)
- **Audio Engine:** Web Audio API + HTML5 Audio
- **PWA:** Vite-PWA with Service Worker
- **Styling:** Vanilla CSS (Glassmorphism design system)
